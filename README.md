# Full Stack Developer Guide - Presentation

## 📁 โครงสร้างโปรเจค

```
Full-Stack-Developer-Guide/
├── index.html              # ไฟล์หลักสำหรับเปิดสไลด์
├── slides/                 # โฟลเดอร์เก็บสไลด์แต่ละหน้า│   
│   └── ... (เพิ่มไฟล์สไลด์ใหม่ที่นี่)
└── README.md              # เอกสารนี้
```

## 🚀 วิธีใช้งาน

1. **เปิดสไลด์**: เปิดไฟล์ `index.html` ในเบราว์เซอร์
2. **นำทาง**: 
   - ใช้ปุ่ม "← ก่อนหน้า" และ "ถัดไป →"
   - หรือกดลูกศร ← → บนแป้นพิมพ์
