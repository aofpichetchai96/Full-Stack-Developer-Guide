Summary Checklist

Full Stack Developer ต้องเข้าใจอะไรบ้าง?

ภาพรวมความรู้ (Knowledge Areas) ที่จำเป็นในปี 2024-2025

System Architecture

  • เข้าใจข้อดี-ข้อเสียของ Monolith vs Microservices
  • รู้จัก MVC, Clean Architecture และ Design Patterns
  • การออกแบบระบบให้ยืดหยุ่น (Scalable Design)

Network & Web

  • เข้าใจ HTTP Methods, Status Codes และ Headers
  • การทำงานของ DNS, CDN, CORS และ WebSockets
  • พื้นฐานการจัดการ Domain และ SSL/TLS

Data Management

  • การออกแบบ Data Modeling และ Relationships
  • เข้าใจ ACID Transactions และ Isolation Levels
  • กลยุทธ์การทำ Caching (Redis) และ Database Indexing

Performance & Scale

  • การทำ Profiling เพื่อหาคอขวด (Bottlenecks)
  • เข้าใจ Horizontal vs Vertical Scaling
  • การ Optimize Frontend (Core Web Vitals) และ Backend

Quality Assurance

  • Testing Pyramid: Unit, Integration, E2E Testing
  • Code Quality tools (Linter, Formatter, SonarQube)
  • กระบวนการ CI/CD เพื่อการส่งมอบที่รวดเร็ว

Security & Compliance

  • ตระหนักถึง OWASP Top 10 vulnerabilities
  • หลักการ Secure Coding และ Data Privacy (GDPR/PDPA)
  • การจัดการ Secrets และ Sensitive Data อย่างปลอดภัย
